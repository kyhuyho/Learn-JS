Bài 301: static method là gì ?
- Dùng để thực hiện 1 chức năng từ class mà ko cần thông qua từ khóa new nếu dùng new để tạo object thì sẽ lỗi

Bài 302: static property là gì ?

Bài 303: Tìm hiểu super và extends
- extend: tạo ra class kế thừa từ class cha
- super: dùng để kế thừa từ constructor của class cha
